Catalogue of block and section templates for the Woody builder.

## :fire: Installation & Usage

First, install Woody Library via the Composer package manager:

```json
{
    "name": "my-project",
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/woody-wordpress/woody-library"
        }
    ],
    "require": {
        "php": "^7.3",
        "woody-wordpress/woody-library": "^1.2.4",
    }
}
```
